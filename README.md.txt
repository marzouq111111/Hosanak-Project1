# 🐎 تطبيق حصانك - Hosanak Horse Services App

## 📱 وصف المشروع
تطبيق شامل لخدمات الخيول في دولة الكويت

### المميزات الرئيسية:
- 🚛 حجز وتتبع نقل الخيول مع GPS
- 🛒 متجر مستلزمات الخيول الشامل
- 🏥 حجز مواعيد الطبيب البيطري
- 🏆 إدارة الفعاليات والمسابقات

## 🛠 التقنيات المستخدمة
- **لغة البرمجة:** Java
- **المنصة:** Android
- **قاعدة البيانات:** Firebase Firestore
- **الخرائط:** Google Maps API
- **الدفع:** KNET (الكويت)

## 👨‍💻 معلومات المطور
- **الاسم:** [ضع اسمك هنا]
- **الكلية:** الكلية التقنية - قسم هندسة الحاسوب
- **المشروع:** مشروع التخرج 2025
- **البلد:** دولة الكويت 🇰🇼

## 📸 لقطات من التطبيق
- صفحة تسجيل الدخول مع دعم اللغتين العربية والإنجليزية
- خريطة تفاعلية تعرض مركبات النقل المتاحة
- متجر مستلزمات مع أسعار بالدينار الكويتي
- نظام حجز مواعيد مع العيادات البيطرية

## 🎯 الهدف من المشروع
حل مشاكل مالكي الخيول في الكويت وتوفير منصة شاملة لجميع احتياجاتهم
